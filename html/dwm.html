1. Preparation

Before you can install dwm, you have to install some programs
(of course you can change the font ttf-ibm-plex, if you want to use icons in the status bar you need to install ttf-font-awesome)
sudo pacman -S xorg-server xorg-xinit xorg-xrandr xorg-xsetroot libxinerama ttf-ibm-plex ttf-font-awesome

2. Download and Installation

Suggestion: create a path in the user directory
mkdir ~/progs
cd ~/progs

2.1 Download the source code

git clone https://git.suckless.org/dwm
git clone https://git.suckless.org/dmenu
git clone https://git.suckless.org/st

2.2 Complie the code

Enter every path to complie the code
If you do not like the st terminal feel free to change that
cd dmenu
sudo make clean install
cd ../dwm
sudo make clean install
cd ../st
sudo make clean install
To start dwm you can type
startx
If you want dwm to start every time after log in you need to config your xinitrc
echo exec dwm >> ~/.xinitrc

3. Configuration

3.1 Github

To configure dwm you can clone our config from Github
cd ~/progs
git clone https://github.com/dib-a/dwm-config.git
now replace the old dwm config with the new one
cd ~/progs
sudo rm dwm/config.h
sudo cp dwm-config/src/configs/dwm.h dwm/config.h
sudo cp dwm-config/src/configs/variables.h dwm/variables.h
sudo rm st/config.h
sudo cp dwm-config/src/configs/st.h st/config.h
sudo cp dwm-config/src/configs/variables.h st/variables.h
