wpa_supplicant

1. Installation

sudo pacman -S wpa_supplicant dhcpcd

2. Configuration

Create a config file in /etc/wpa_supplicant/
sudo vim /etc/wpa_supplicant/home.conf
The config should look like this
ctrl_interface=/run/wpa_supplicant
ctrl_interface_group=wheel
update_config=1
ap_scan=1
network={
ssid="SSID of the network"
psk="password"
}

3. Run

Replace wlan0 with your interface
sudo wpa_supplicant -B -i wlan0 -c /etc/wpa_supplicant/home.conf
Then run
sudo dhcpcd wlan0
